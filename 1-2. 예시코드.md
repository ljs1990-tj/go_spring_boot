== application.properties ==
```properties
# 식당 세부 도면 파일: src/main/resources 폴더 안에 위치합니다.

# 1. 출입문 번호 설정 (다른 프로그램과 충돌하지 않게 8081 권장)
server.port=8081

# 2. 🌟 View Resolver (화면 경로 설정) 🌟
# 서빙 직원이 JSP를 찾을 때 앞뒤로 붙여줄 주소를 고정합니다.
# 주의: 실제로 프로젝트 내에 src/main/webapp/WEB-INF/views/ 폴더를 만드셔야 합니다!
spring.mvc.view.prefix=/WEB-INF/views/
spring.mvc.view.suffix=.jsp

# 3. 데이터베이스(냉장고) 연결 설정 (본인의 DB 아이디/비밀번호로 수정하세요)
spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver
spring.datasource.url=jdbc:mysql://localhost:3306/my_database?serverTimezone=Asia/Seoul
spring.datasource.username=root
spring.datasource.password=1234
```

== MainController.java ==
```java
// 식당의 홀 매니저 파일: 적절한 패키지(예: com.example.myproject.controller) 안에 생성합니다.
package com.example.myproject.controller;

import org.springframework.stereotype.Controller;
import org.springframework.web.bind.annotation.GetMapping;

// @Controller: 스프링에게 "이 클래스는 홀 매니저야!" 라고 알려주는 명찰입니다.
@Controller
public class MainController {

    // 사용자가 주소창에 기본 주소(localhost:8081/)로 접속했을 때 실행됩니다.
    @GetMapping("/")
    public String showMainPage() {
        // "index"라는 이름을 반환하면, View Resolver가 앞뒤로 경로를 붙여서
        // 최종적으로 /WEB-INF/views/index.jsp 화면을 손님에게 보여줍니다.
        return "index"; 
    }
}
```
